# AI Risk Supervision Playbook

A polished client-only MVP for financial regulators, central bank supervisors, prudential supervisors, conduct supervisors, technology risk supervisors, and policy analysts assessing AI risk management at supervised financial institutions.

The app asks structured questions and generates a supervisory playbook with:

- supervisory intensity
- transparent risk score
- top risk drivers
- AI profile summary
- evidence request list
- interview questions by stakeholder group
- control checklist
- red flags
- recommended supervisory next steps
- copyable Markdown report

## Stack

- Vite
- React
- TypeScript
- plain CSS
- Vitest

## Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm test
npm run build
npm run lint
```

## Scope

Client-only. No backend, auth, database, persistence, file upload, payments, dashboard, or AI API calls.

This tool is not legal advice and does not replace supervisory judgment. It is a structured planning aid.
